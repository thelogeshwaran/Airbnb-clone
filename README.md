# Airbnb-clone

It is a Airbnb clone using React Native and AWS amplify.

## Features

* It has a google Autocomplete feature, which helps you to search fastly.
* Using Maps feature you can choose the hotels from maps.
* Filters the result based on guests and location.
* Gives you the complete details of the hotel.


## Technologies
* **Frontend:** React Native
* **Styling:** CSS and React Vector Icons
* **Database:** Amplify
* **Authentication:** Amplify
